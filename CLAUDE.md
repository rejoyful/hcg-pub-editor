# hcg-pub-editor — 작업 컨텍스트

## 제품 정의
학지사그룹(HCG) 서적 조판용 웹 에디터.
원고(HWPX/DOCX/PDF/HTML/MD/TXT/IDML/XML 및 복붙 텍스트)를 입력받아 
책 조판 형식(용지·서체·단·머리말 등)으로 편집·내보내기 한다.

## 핵심 파이프라인
원고 입력 → ①포맷 인식 → ②문서 파싱 → ③이미지 추출·배치 
         → ④스타일 정규화 → ⑤에디터 적용 → (키워드 자동분류) → 편집 → 내보내기

## 절대 규칙
1. "AI 자동 스타일"은 검증·제안 전용이다. 본문 텍스트를 AI가 직접 수정하지 않는다.
2. 기존 키워드 기반 분류 규칙을 임의로 변경하지 않는다. 확장만 한다.
3. 내보내기(Word/PDF/HTML)는 에디터 프리뷰와 1:1 시각 일치를 목표로 한다.
4. 이미지는 에디터에서의 위치·크기·정렬이 내보내기에서도 동일하게 유지돼야 한다.
5. 작업은 PR 단위로 분리한다. 한 PR에 한 과제만.

## 분류 체계
- 영역: 앞부속(Front) / 본문(Body) / 뒷부속(Back)
- 앞부속 유형: 속표지 / 판권지 / 커버디자인 / 저자 소개 / 머리말 / 목차
- 본문 유형: 장 표지(chapter_opener) / 본문
- 뒷부속 유형: 참고문헌 / 색인 / 에필로그

## 테스트 원고
- 샘플: "화타를 꿈꾸며" (화타_앞부속.pdf + 화타01_06장.pdf)
  · 앞부속: 속표지·판권지·커버·저자 소개·머리말·목차
  · 본문: 1~6장(장 번호 + 장 제목 형식)
- 회귀 테스트 시 이 원고로 검증.
