HCG Pub 데모 사용법

이 데모는 Vivliostyle Pub 계열의 웹 조판 에디터 구조를 그대로 구현한 단일 파일 프로토타입입니다. 왼쪽에 Markdown을 입력하면 오른쪽에 책 레이아웃으로 실시간 조판됩니다.

  • 상단 테마 / 판형 / 본문 크기를 즉시 전환 가능
  • 샘플 원고로 빠르게 체험 가능(한국어·영어 혼용 포함)
  • PDF 내보내기는 브라우저 인쇄 대화창에서 ‘PDF로 저장’ 선택
  • 한글 금칙(word-break: keep-all) 적용되어 단어 중간에서 끊기지 않음
  • 본 데모는 단일 HTML 파일로 동작하며, 모든 처리가 로컬 브라우저에서만 일어남(원고가 외부로 전송되지 않음)

실제 서비스화 시 필요한 기능: 다중 파일 원고 관리, Git 연동, 실시간 협업(공저자), 댓글/트랙체인지, HWPX 입력 변환, InDesign 최종 조판 연동.
